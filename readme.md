# FRETO

## Descrição do projeto
O FRETO é uma plataforma moderna de logística que conecta clientes que precisam realizar transportes ou mudanças com motoristas parceiros disponíveis. O grande diferencial do sistema é o uso de processamento assíncrono (filas de mensageria) para realizar o match entre cliente e motorista em background, garantindo uma experiência de usuário fluida e responsiva.

## Tecnologias

### Front-End
- React com TypeScript
- Vite
- Bootstrap

### Back-End
- Java 21
- Spring Boot 3
- Spring Web
- Spring Security
- Spring Data JDBC
- Spring AMQP (RabbitMQ)
- Spring Mail
- Spring Validation
- Project Lombok
- Banco de Dados: Azure SQL Database

### DevOps
- Docker
- Github actions (CI/CD)

### Arquitetura
- Arquitetura Orientada a Eventos e Micro-Serviços
- Processamento Assíncrono com mensageria (RabbitMQ)
